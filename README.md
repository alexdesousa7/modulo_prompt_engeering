Este repositorio recopila las **prácticas, ejercicios y notebooks** realizados durante el módulo de **Prompt Engineering** del Máster de Inteligencia Artificial de **Pontia**.

Incluye ejemplos reales, experimentos y técnicas aplicadas con modelos de lenguaje modernos (Gemini, Groq, Hugging Face, etc.), siguiendo las metodologías vistas en el módulo.

---

## Contenido del repositorio

- **Ejercicio_1/**  
  Notebooks y datasets del primer ejercicio práctico (Airbnb reviews, análisis, prompting aplicado).

- **1-PromptEngineering/**  
  Materiales y ejemplos del módulo: principios, técnicas, iteración, ampliación y transformación de prompts.

- **gemini/**, **groq/**, **huggingface/**  
  Pruebas y ejemplos usando diferentes proveedores de modelos generativos.

---

## Objetivo

Este repositorio sirve como:

- Base de prácticas del módulo de Prompt Engineering  
- Material de estudio y referencia  
- Espacio para experimentar con técnicas de prompting  
- Registro del progreso durante el Máster  

---

## Requisitos

Para ejecutar los notebooks:

- Python **3.10+**  
- Entorno virtual (recomendado)  
- Dependencias instaladas según cada notebook  
- Archivo `.env` local con claves API necesarias (no incluido en el repo)

---

## Tecnologías utilizadas

- **Python**  
- **Jupyter Notebooks**  
- **Gemini API**  
- **Groq API**  
- **Hugging Face Inference**  
- **Técnicas de Prompt Engineering** (Chain-of-Thought, Iterative Refinement, Role Prompting, etc.)
